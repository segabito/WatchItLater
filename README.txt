WatchItLater.js

■どんなスクリプト？

「ニコニコ動画(9)のマイリスト登録を(β)と同じにしたい！」という所から開発が始まり、
(9)からQまでやってきました。

ニコニコ動画の視聴ページを開く前にマイリストに登録するGreasemonkeyスクリプトです。
FirefoxとChromeに対応しています。

配布ページ
http://userscripts.org/scripts/show/151269 
開発版のデモ動画(ちょっと古い)
http://www.nicovideo.jp/watch/sm18845030 (基本機能紹介)

ランキングや検索結果を見て「これは面白そうだなあ」「あとで見よう」という動画があった時、
とりあえずマイリストにどんどん登録しておいて、あとからZeroWatchやVitaやiPadで
まとめて消化するのに便利です。

「ReadItLaterのニコニコ版」という事で「WatchItLater」という名前をつけました。



■その他の拡張機能
・動画の視聴中もいつでもマイリスト登録出来る
　ポップアップウィンドウ不要

・視聴ページのヘッダ部分に再生数やマイリスト数表示

・動画を開く前にタグを取得できる
　・当然そこからタグ検索に飛べる

・Qwatchの動画検索でコメントやシークが可能。 画面サイズも可能な限り最大化

・動画再生開始時に、自動的に動画が見やすい位置にスクロール
　説明文の長い動画で重宝します

・または、再生開始時に自動最大化 (プレイリストも自動で閉じます)
　・そして動画終了時に最大化解除 (原宿プレイヤーと同じ挙動です)

・背景の空白部分をダブルクリックすると、動画の位置に自動スクロール
　市場を下まで見てから動画に戻りたい時など、地味にすごく便利。

・とりあえずマイリストからの削除機能

・とりあえずマイリストに登録済みだった場合、通常はエラーが表示されるが、
　このスクリプトではエラーは出さず、とりあえずマイリストの先頭に移動する
　(「すでに登録済です」と言われても、数百件の中から探すのがダルいので)

・コメントパネルを広くする機能
　コメントパネル内を横スクロールさせなくても
　動画の何秒目にコメントされたのか何月何日にコメントされたのかがわかる幅になる

・左のパネルにサムネと説明文を表示

・プレイリスト部分の開閉機能 (デフォルトは閉)

・画面の縦スクロールをなるべく減らすための、数々の細かい細工

・「@ジャンプ」キャンセラー

・タグが2行以内の時、自動で高さを調節して縦解像度を節約する

・ニコニコニュースの履歴を表示

・Qwatch上からお気に入りタグ検索

・Qwatch上からお気に入りマイリスト表示
　これで気になるシリーズの新着が素早く確認できる！

・Qwatch上から「あなたにオススメの動画」表示

・Qwatch上から動画ランキング表示

・「ウィンドウがアクティブの時だけ自動再生する」機能を追加

・タグ検索・マイリスト検索のソートを記憶するようにした


■開発の方針
・自分の欲しいと思った機能をひたすら追加する
・元からある機能をデフォルトで隠す事はあっても、殺したりはしない
・広告はノータッチ

(9)の頃から勉強しながら少しずつ継ぎ足されていったコードなので、統一感がなくて汚いです。
